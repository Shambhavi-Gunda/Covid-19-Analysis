# Covid-19-Analysis
 Compare recovery rates between countries and Identify signs of effective healthcare systems


This project analyzes global COVID-19 data to explore how different countries performed in terms of recovery. It focuses on identifying which countries had strong recovery rates and which improved the fastest over time.



## Dataset

- Source: [Kaggle - COVID-19 Clean Dataset](https://www.kaggle.com/datasets/imdevskp/covid-19-corona-virus-india-dataset)
- Key columns used: `Date`, `Country/Region`, `Confirmed`, `Recovered`

## What Was Done

1. **Overall Recovery Rate**  
   Calculated as `Recovered / Confirmed` for each country to see how well people recovered.

2. **Recovery Rate Over Time**  
   Measured how quickly recovery rates improved over time using a basic trend analysis.

3. **Visualizations**  
   - Bar charts showing top countries by recovery rate  
   - Charts showing countries improving the fastest

## Tools Used

- Python
- Pandas
- Matplotlib
- Google Colab

## How to Run

- Upload `archive.zip` in Google Colab
- Extract and load the CSV
- Run the code cells to view analysis and plots

## Summary

This project is a simple exploration of real-world data to understand how different countries managed COVID-19 recoveries. It combines data cleaning, basic metrics, and visual storytelling to draw insights.
